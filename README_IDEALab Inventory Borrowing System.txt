IDEALab Inventory Borrowing System

This application IS developed using MIT App Inventor and is used to replace manual inventory borrowing systems used in IDEALab.

Users can:
- Login using name and roll number
- Select components from available inventory
- Enter borrowing quantity
- Borrow components

The borrowing data is stored in Google Sheets, which acts as a
backend database through Google Apps Script integration.

Technologies Used:
- MIT App Inventor
- Google Sheets
- Google Apps Script
